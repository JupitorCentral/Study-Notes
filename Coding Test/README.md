## 시간복잡도 + 값의 범위

문제를 풀때 시간복잡도 먼저 계산한다.
대략 1초 = 1억번 - 1e9 연산

값의 범위를 생각한다. 
int -> 2^31 - 1 ~ -2^31
long -> 2^63 - 1 ~ -2^63

값을 비교할때 범위를 초과하지 않기 위해서, 
a - b < 0 이라면, a < b 임을 이용해야 할 때가 있다.
