

### 이차원 배열

``` java
int[][] arr = new int[10][10];
```
이렇게 선언하면 10 * 10 크기의 배열이 생성되고, 모든 요소는 0으로 초기화된다.

### 가변 배열 (ragged array)

``` java
int[][] arr = new int[10][];
```
이렇게 선언하면 10 크기의 배열이 생성되고, 모든 요소는 null 로 초기화된다.
이를 사용하기 위해서는 각 행에 대한 배열을 따로 생성해야 한다.

``` java
arr[0] = new int[10];
arr[1] = new int[20];
arr[2] = new int[30];
```
